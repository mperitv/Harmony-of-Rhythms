# Harmony of Rhythms  
One day, the rhythm of the celestial clock in heaven breaks down, and heaven begins to collapse. A human chosen by God is tasked with repairing this clock. Our character passes through purgatory to reach heaven and attempts to repair the clock along with its rhythm! (Currently only the Purgatory part exists) This is my first game I've ever made!!! so I struggled a lot while making it, but I finally managed to create something. I hope you like it too. Since it's still very very early the assets aren't complete yet.

## Controls
D for Red Bats  
F for Blue Bats  
J for Yellow Bats  
K for Green Bats  
Space for Purple Bats
